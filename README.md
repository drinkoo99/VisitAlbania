# Visit Albania

## Overview
**Visit Albania** is a tourism website that showcases the natural beauty, activities, traditional food, and cultural events of Albania. The goal is to provide an engaging and interactive experience for users, fully responsive on all devices.

## Progress So Far
- Navigation bar with dropdown menus for Destinations and Activities.
- Hero section with a carousel of 7 background images, zoom out effect, and arrow navigation.
- Smooth animations for hero text and navigation links.
- Responsive design with a hamburger menu for mobile screens.
- Navbar style changes on scroll.
