# Visit Albania

## Overview
**Visit Albania** is a tourism website that showcases the natural beauty, unique destinations, activities, traditional food, and cultural heritage of Albania. The primary goal is to deliver an engaging and interactive experience for users, fully responsive across all devices.

## Progress So Far
- Navigation bar with dropdown menus for Destinations and Activities.
- Hero section with a carousel of 7 background images, zoom-out effect, and arrow navigation.
- Smooth animations for hero text and navigation links.
- Fully responsive design with a hamburger menu for smaller screens.
- Navbar style transitions on scroll.
- **Destinations section on the main page is now completed and fully responsive** (Beaches, Mountains, Rivers, Lakes, Countryside, Historic Cities, National Parks).
  - Users will soon be able to explore each destination in dedicated pages.
